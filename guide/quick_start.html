# 快速开始

# 环境准备

  1. python 3.7以上 (opens new window)版本环境
  2. unrpyc (opens new window)文件 (用于反编译rpyc文件)
  3. unrpa文件 【安装完python以后以及配置好环境变量(Path路径)请使用 pip install unrpa 安装unrpa包】(用于反编译rpa文件)
  4. 以上3点可用unren (opens new window)工具一次性解决(以汉化，如果打开闪退请使用封装版 (opens new window))，直接放入游戏根目录根据提示执行即可【目前unren不适用于一些作者进行简单加密的游戏，建议使用unrpyc+unrpa】
  5. ren'py sdk (opens new window) (本人使用的是renpy-7.4.11版本)
  6. 汉化包所描述的游戏
